#### المتغيرات: 

ماهي المتغيرات؟

هي حجز مساحة في ذاكرة الوصول العشوائي ليتم وضع القيم فيها، ويتم تحديد مقدار المساحة التي سيتم حجزها حسب نوع المتغير.



كيف يتم تعريف المتغيرات؟

```python
a = 1
b = "Hello"
c = 'Hello'
d = 2.34
e = 3 + 4j
```



ماهي شروط تسمية المتغيرات؟

1. كل المتغيرات يجب ان تبداء بحرف او شارحة سفلية
2. لايمكن ان تبدأ اسماء المتغيرات برقم
3. اسماء المتغيرات يجب ان تتكون من حروف وارقام وشارحة سفلية فقط
4. المتغيرات في بايثون حساسة للأحرف

#### أنواع البيانات:

ماهي انواع المتغيرات؟

المتغيرات النصية `String`

المتغيرات العددية `Integer, Float, Complex`

المتغيرات المنطقية `bool`

متغيرات القوائم `list, tuple, range`

المجموعات `Set, FrozenSet`

المتغيرات الثنائية `byte, byteArray, memoryView`



كيف يتم اظهار نوع المتغير؟

```python
s = "Hello"
print(type(s))
```
